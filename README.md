A custom Home Assistant integration that supercharges your PlayStation 4 experience! It adds missing features from the official integration.

## ✨ Features
✅ **Controller Battery Monitoring:** View your DualShock 4 battery level directly in Home Assistant.
✅ **Advanced Game Detection:** Know exactly which game is currently playing.
✅ **Rich Notifications:** Send detailed notifications to your PS4 screen.
✅ **Automations:** Trigger automations when specific games launch.

## 📥 Installation

Please refer to the [Installation Guide](docs/installation.md) in the `docs` folder for detailed instructions on how to add this to HACS.
